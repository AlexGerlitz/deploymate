# DeployMate v0.1.0

Initial public release candidate for hiring and technical review.

## What This Release Shows

- a working self-hosted deployment control panel with a live demo at `https://deploymatecloud.ru`
- a clear split between product surface, operational tooling, and release discipline
- evidence of production thinking: preflight checks, smoke coverage, runbooks, and safer deployment defaults

## Highlights

- deployment dashboard with runtime operations and deployment detail pages
- server registry with connection tests, diagnostics, and suggested free ports
- deployment templates with preview, filters, and usage tracking
- admin users workspace with saved views, audit, exports, and bulk actions
- admin upgrade inbox with saved views, audit, exports, and bulk actions
- backup bundle export and restore dry-run analysis
- public trial signup flow with non-admin restrictions
- remote-only production profile without default Docker socket access

## Engineering Signals

- documented architecture, production, security, and release flow
- dedicated frontend smoke coverage for admin and auth surfaces
- scripted preflight, post-deploy smoke, and release workflow helpers
- clean public repository structure intended for reviewer navigation

## Recommended Reviewer Path

1. open `https://deploymatecloud.ru/register`
2. create a trial account
3. inspect `/app`
4. inspect `/deployments/[deploymentId]`
5. inspect `/app/users`
6. inspect `/app/upgrade-requests`
